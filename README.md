# Car Price Predictor

## Overview
Car Price Predictor is a Django-based web application that predicts the estimated selling price of a used car.
The prediction is generated using a machine learning model trained on real-world used car data.

This project helps users get an approximate resale value of a car based on important features such as brand, year, fuel type, transmission, and more.

---

## Key Features
- Predicts the price of used cars
- Simple and user-friendly web interface
- Machine learning model integrated with Django backend
- Supports multiple car attributes for better estimation
